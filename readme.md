# GodotSteam's Skillet

Designed to showcase the features of GodotSteam; packing in as many of the Steam classes as possible.  The game being shared as open-source will allow developers to see how things work in Steamworks and Godot with real-world examples. Users may copy and modify the code for their own needs and games.  Since it is meant to be educational, it will be designed in a somewhat simplified way to accommodate all levels.

Obviously, not all of Skillet's code is available yet. Scripts and scenes will be added when they are deemed "completed enough" for public use.

[You can find it here, though the store page is not quite live yet.](https://store.steampowered.com/app/3013040/Skillet/)  The game will be released in December.

---

## Game Modes

The kitchen is your playground. Be the last player standing in one of five game modes:

**Fire-Water-Sponge**

Fire burns sponge, sponge absorbs water, water douses fire. You get how this works.

**Gumbo**

A little bit of everything and that is really just too much.

**Hot Potato**

The floor is lava. Well, boiling hot oil. Jump for every melting platform you can while you can.

**Pass The Torch**

The Match is "it" and coming for you. Get tagged to become yet another Match, spreading the flames.

**Trial By Fire**

Avoid being ignited by various hazards while knocking friends into said hazards. Accidentally.

## Cosmetics / Skins

Cosmetics and skins are all available as Inventory items that can be traded with friends; sorry, no marketable items!  If you have Godot in your library on Steam, the game unlocks a special skin / cosmetic; and it checks for the following Godot-made games too:

- [Resolutiion](https://store.steampowered.com/app/975150/Resolutiion/)
- [Lila's Sky Ark](https://store.steampowered.com/app/1573390/Lilas_Sky_Ark/)
- [Satryn Deluxe](https://store.steampowered.com/app/1677360/satryn_deluxe/)
- [WindowKill](https://store.steampowered.com/app/2726450/Windowkill/)
- [Drift](https://store.steampowered.com/app/2159650/Drift_Space_Survival/)
- [Brotato](https://store.steampowered.com/app/1942280/Brotato/)
- [Chillquarium](https://store.steampowered.com/app/2276930/Chillquarium/)

## Extra Content / DLC

Since Skillet is free-to-play, we will be offering two DLC packs: Godot Dinner Party and GodotSteam Snack Pack. Of course, [any donations through Github Sponsors goes towards Skillet's development and the general GodotSteam development.](https://github.com/sponsors/Gramps)

**Godot Dinner Party**

This DLC contains Godot-themed skins and cosmetics as well as other Godot-made games' cosmetics. All proceeds from this DLC go to the Godot Foundation after Valve's cut. [You can also donate to Godot directly here.](https://fund.godotengine.org)

**GodotSteam Snack Pack**

This DLC contains additional skins and cosmetics based on GodotSteam and other creations.
